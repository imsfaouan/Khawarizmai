**OSINT** هي عملية جمع وتحليل المعلومات المتاحة للعموم — عبر الإنترنت، السجلات العامة، ووسائل الإعلام — لإنتاج استخبارات مفيدة. تُستخدم في الأمن السيبراني، التحقيقات الصحفية، والبحوث الأمنية.

**Robin** هي أداة OSINT مدعومة بالذكاء الاصطناعي متخصصة في **الدارك ويب**. تعطيها هدفاً (شخص، دومين، أو IP) فتبحث بعمق في محركات بحث الدارك ويب، تفلتر آلاف النتائج، وتعطيك ملخصاً ذكياً ومنظماً — مع دعم كامل للنماذج المحلية مثل Ollama لحماية خصوصيتك.

## مقدمة

في عالم الأمن السيبراني والبحث الاستخباراتي، أصبح جمع المعلومات (OSINT) من أهم المهارات.

لكن المشكلة دائمًا هي نفسها:
- كيف تبحث في الدارك ويب بأمان دون المرور بخدمات سحابية تسرّب بياناتك؟
- كيف تحصل على ملخص ذكي ومنظم من آلاف الصفحات؟
- وكيف تفعل كل هذا بخصوصية كاملة؟

الحل الذكي هو أداة **Robin**.

Robin ليست مجرد أداة بحث — هي مساعد ذكي يستخدم نماذج الذكاء الاصطناعي المحلية ليقرأ، يفلتر، ويلخص المعلومات نيابةً عنك، مع دعم كامل للبحث في الدارك ويب عبر شبكة Tor.

## ما هي أداة Robin؟

Robin هي أداة مفتوحة المصدر متخصصة في **جمع المعلومات (OSINT) من الدارك ويب** باستخدام الذكاء الاصطناعي.

وصل المشروع إلى **4,400 نجمة على GitHub** و**855 fork** — دليل على اهتمام حقيقي من مجتمع الأمن السيبراني.

تعطيها هدفاً فتقوم تلقائياً بـ:
- البحث العميق في محركات بحث الدارك ويب عبر Tor
- تشغيل استعلامات OSINT ذكية ومحسّنة بالذكاء الاصطناعي
- فلترة آلاف النتائج واستخراج المفيد منها
- تلخيص المعلومات بطريقة ذكية ومنظمة
- حفظ تقرير كامل للتحقيق

## المتطلبات الأساسية قبل التثبيت

**⚠️ شرط أساسي لا يمكن تخطيه: Tor**



Robin تحتاج **Tor** للوصول إلى الدارك ويب — بدونه لن تعمل.

**تحميل متصفح TOR :**
[TOR](https://www.torproject.org/download/)

**تثبيت Tor:**

# Linux / Windows (WSL)
apt install tor

# Mac
brew install tor


بعد التثبيت تأكد أن Tor يعمل في الخلفية قبل تشغيل Robin.

**باقي المتطلبات:**
- Docker (مُوصى به) أو Python 3.10+
- مفتاح API لأحد النماذج — أو Ollama للتشغيل المجاني المحلي

**تحميل متصفح Docker :**
[Docker Desktop](https://www.docker.com/products/docker-desktop/)

## طريقة التثبيت عبر Docker (الأسهل والأنظف) 


**الخطوة 1 — أنشئ ملف .env:**

انسخ الملف النموذجي من المستودع وأضف مفتاح API الخاص بك:

OLLAMA_BASE_URL=http://host.docker.internal:11434


**الخطوة 2 — حمّل صورة Robin:**

docker pull apurvsg/robin:latest


**الخطوة 3 — شغّل Robin:**

docker run --rm \
  -v "$(pwd)/.env:/app/.env" \
  --add-host=host.docker.internal:host-gateway \
  -p 8501:8501 \
  apurvsg/robin:latest


**الخطوة 4 — افتح المتصفح:**

http://localhost:8501


ستظهر لك واجهة ويب بسيطة وتفاعلية جاهزة للاستخدام.
