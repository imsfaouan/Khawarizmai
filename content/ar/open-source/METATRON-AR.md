**METATRON** هو مساعد اختبار اختراق ذكي يعمل **100% محليًا** على لينكس بدون إنترنت وبدون أي API. تعطيه هدف (IP أو موقع) فيقوم بتشغيل أدوات مثل nmap وwhois، ثم يحلل النتائج بالذكاء الاصطناعي ويقترح الثغرات والحلول تلقائيًا. مثالي للمبتدئين والمحترفين في الأمن السيبراني.

---

## مقدمة

في عالم الأمن السيبراني، كان اختبار الاختراق (Penetration Testing) دائمًا يتطلب أدوات متعددة، خبرة عالية، وغالباً اتصال بالإنترنت.

اليوم، تغير الأمر.

**METATRON** هو أداة جديدة تعمل كمساعد ذكي محلي 100%، يجمع بين أدوات الاختراق التقليدية والذكاء الاصطناعي في أداة واحدة تعمل بدون إنترنت وبدون أي اشتراك أو API حاليا .

**في هذا المقال سأشرح لك:**
- كيف يعمل METATRON بالضبط
- طريقة التثبيت خطوة بخطوة
- المميزات والاستخدامات
- التحذيرات القانونية المهمة

## كيف يعمل METATRON؟

METATRON يعتمد على نموذج ذكاء اصطناعي محلي (local LLM) يعمل مع أدوات pentest التقليدية:

1. تعطيه هدفًا (IP address أو domain)
2. يقوم تلقائيًا بتشغيل أدوات مثل:
   - nmap
   - whois
   - subfinder
   - وغيرها
3. يحلل النتائج بالذكاء الاصطناعي
4. يقترح الثغرات المحتملة، طرق الاستغلال، والحلول الموصى بها
5. يطلب عمليات إضافية إذا لزم الأمر (مثل فحص منفذ معين)

كل هذا يحدث **داخل جهازك** بدون إرسال أي بيانات إلى الخارج.

![AIhacking](https://github.com/sooryathejas/METATRON/blob/main/screenshots/scan_running.png)

## المميزات الرئيسية

- يعمل 100% offline (بدون إنترنت)
- تحليل ذكي باستخدام نموذج AI محلي
- حفظ كل النتائج في قاعدة بيانات محلية
- يدعم فحص شامل ومتكرر
- لا يحتاج API أو اشتراكات
- مناسب للمبتدئين والمحترفين في الأمن السيبراني

## طريقة التثبيت (لينكس فقط)

METATRON يعمل حاليًا على **لينكس** فقط.

## الخطوات البسيطة:

*1. افتح الـ Terminal : *
أي Terminal عادي في توزيعة لينكس (مثل Ubuntu، Kali Linux، أو Parrot OS).
 (Ctrl + Alt + T في معظم التوزيعات).

لكن الأفضل والأكثر أمانًا هو استخدام Kali Linux أو Parrot OS،

*2. انسخ الأمر التالي:*

git clone https://github.com/sooryathejas/METATRON.git
cd METATRON

*أنشئ بيئة Python نظيفة (مهم لتجنب المشاكل):*

python3 -m venv venv
source venv/bin/activate

*قم بتثبيت المتطلبات:*

pip install -r requirements.txt

*قم بتثبيت بعض الأدوات النظامية المطلوبة:*

sudo apt update
sudo apt install nmap whois whatweb curl dnsutils nikto

بعد الانتهاء من هذه الخطوات

شغل الأداة:

python metatron.py

**ملاحظة 1:** يُفضل استخدام جهاز لينكس قوي (مثل Ubuntu أو Kali Linux) مع ذاكرة RAM لا تقل عن 16 جيجا للحصول على أفضل أداء.

**ملاحظة 2:** اذا كنت مبتدئا من صفر يمكنك استخدام الذكاء الاصطناعي مثل *claude* او أي ذكاء اصطناعي تفضله  ليساعدك على تتبيث الأداة METATRON

**في ماذا تستخدم METATRON؟**

اختبار أمان شبكتك الخاصة أو مشاريعك
تعلم الـ Ethical Hacking بشكل عملي
إجراء فحوصات أولية سريعة قبل استخدام أدوات متقدمة
تحليل الثغرات في بيئات المختبر (Lab)

**تحذير قانوني هام**
METATRON أداة قوية وهجومية.

استخدامها على أنظمة أو مواقع بشكل غير قانوني وقد يعرضك للمساءلة القانونية.
