## Qu’est-ce que Sniffnet ?

Sniffnet est un outil open source conçu pour **surveiller le trafic internet** de manière confortable et simple.

Au lieu d’utiliser des outils complexes comme Wireshark, Sniffnet vous offre une interface moderne et intuitive qui vous montre :
- Toutes les connexions actives en ce moment
- Quelle application ou quel appareil consomme de l’internet
- Les protocoles et services utilisés
- Le volume de données envoyées et reçues
- Les sites web et serveurs avec lesquels vos appareils communiquent
- **La localisation géographique** des hôtes distants (une fonctionnalité très puissante)

Tout cela vous aide à détecter les applications qui « vous espionnent » ou envoient vos données sans votre connaissance, et à savoir qui vole votre internet.

## Principales fonctionnalités de Sniffnet

- Fonctionne sur **Windows, macOS et Linux**
- Interface moderne et très simple (Intuitive)
- Surveille le réseau en temps réel
- Affiche des graphiques de consommation internet
- Détecte plus de 6000 protocoles et services
- Supporte le filtrage avancé
- **Export et import de fichiers PCAP** (fonctionnalité professionnelle)
- 100% gratuit et open source
- Actuellement plus de **33 200 étoiles** sur GitHub
- Le projet est **actif et évolue en continu** (dernière version novembre 2025)

## Méthode d’installation (Très facile)

Sniffnet est l’un des outils les plus faciles à installer, comme n’importe quel programme classique :

### Sur Windows :
- Téléchargez le fichier `Sniffnet_Windows_x64.msi`
- Exécutez-le et suivez les étapes simples

### Sur macOS :
- Téléchargez le fichier `.dmg` approprié (Intel ou Apple Silicon)
- Glissez-le dans le dossier Applications

### Sur Linux :
- Disponible en AppImage, DEB ou RPM selon votre distribution
