## Qu'est-ce que Pinokio ?

**Pinokio** est un navigateur intelligent conçu spécifiquement pour installer et exécuter des outils IA open source — en un seul clic.

Au lieu d'ouvrir un Terminal et de taper des commandes, ou d'installer Python et Git et de gérer des problèmes de compatibilité — Pinokio s'occupe de tout cela en coulisse. Vous ne voyez qu'un bouton "Install", puis "Run".

Le projet a atteint **7 300 étoiles sur GitHub** — et continue de croître avec chaque nouvel outil IA qui apparaît.

## Comment Fonctionne Pinokio ?

L'idée est simple et brillante à la fois :

Pinokio contient tout ce dont vous avez besoin — Python, Git, Conda, pip, npm et d'autres outils — tous installés et isolés dans un seul dossier sur votre machine `~/pinokio`.

Quand vous appuyez sur Install pour n'importe quel outil :
- Pinokio télécharge automatiquement le code depuis GitHub
- Installe tous les prérequis dans un environnement isolé
- Lance l'outil sans affecter les autres programmes de votre appareil

Et quand vous supprimez un outil — tous ses fichiers disparaissent complètement sans aucun résidu.

## Installation — Seulement 3 Étapes

**Étape 1 — Téléchargez Pinokio :**

Allez sur la page des releases et téléchargez la version adaptée à votre système :
[Pinokio v7.2.0 Releases](https://github.com/pinokiocomputer/pinokio/releases/tag/v7.2.0)

**Pour Windows :**
Téléchargez le fichier `.exe` et exécutez-le directement

**Pour Mac :**
Téléchargez le fichier `.dmg` (Intel ou Apple Silicon)

**Pour Linux :**
Téléchargez le fichier `.AppImage` ou `.deb` selon votre distribution

**Étape 2 — Ouvrez Pinokio :**
Après l'installation, ouvrez l'application directement — vous trouverez une page d'accueil simple.

**Étape 3 — Choisissez un outil et installez-le :**
Allez dans l'onglet **Discover** et cherchez l'outil que vous voulez — appuyez sur Install et attendez.

C'est tout. Pas de Terminal, pas de Python, pas de problèmes.

## Sécurité — Comment Vous Protéger ?

Pinokio est conçu avec la sécurité dès le départ :

**Programmes isolés** — Tout est dans `~/pinokio`, ne touche pas le reste de votre système.

**Scripts open source** — Chaque script est écrit en JSON et vous pouvez le lire avant de l'exécuter.

**Page Discover sécurisée** — Les programmes affichés y sont personnellement revus par l'équipe avant d'être ajoutés.

**Protection Supply Chain** — Empêche l'installation de packages publiés depuis moins de 72 heures pour éviter les nouvelles vulnérabilités.

> **Avertissement :** N'installez pas de scripts provenant de sources externes non fiables. Restez sur la page Discover pour une sécurité complète.
