## Questions Fréquemment Posées (FAQs)

**1. Ces API sont-elles vraiment 100% gratuites ?** La plupart offrent un "Free Tier" généreux avec un quota mensuel de requêtes. Certaines sont totalement gratuites et sans limites.

**2. Ai-je besoin d'une clé API (API Key) ?** Oui, la majorité demande une inscription gratuite pour obtenir une clé (indiqué par *Auth: apiKey* dans la liste).

**3. Que faire si une API s'arrête ?** Le dépôt est maintenu par la communauté, mais vérifiez toujours la date de dernière mise à jour de l'API spécifique avant de l'intégrer en production.

**4. Puis-je les utiliser pour un projet commercial ?** Vérifiez la licence de chaque API. Beaucoup autorisent l'usage commercial sous certaines conditions.
