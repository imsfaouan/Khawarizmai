## How to Run Gemma 4 on Your Phone Without Internet

## 1. On Android and iOS (The Easiest Way)

![gemma4](https://res.cloudinary.com/dnavgirzt/image/upload/v1775396919/ee50ad51-68c9-4675-9567-09e6909f7712_lacmea.jpg)

1. Download the **Google AI Edge Gallery** app:
   - Android: [Google Play Link](https://play.google.com/store/apps/details?id=com.google.ai.edge.gallery)
   - iOS: [App Store Link](https://apps.apple.com/us/app/google-ai-edge-gallery/id6749645337)

2. Open the app
3. Choose the mode (Chat / Image / Audio)
4. Download the Gemma 4 model:
   - E2B → Suitable for mid-range phones
   - E4B → More powerful (requires a stronger phone)
5. Start chatting immediately — everything works offline.

## How to Run Gemma 4 on Computer (Windows & Mac)

## For Beginners on Windows (Easiest Method in 2026)

![-windowsgemma4](https://res.cloudinary.com/dnavgirzt/image/upload/v1775397381/image_2026-04-05_145618115_aop3gm.png)

2. Open LM Studio
3. Search for "Gemma 4" or "Gemma-4-E4B"
4. Download the model
5. Run it and start chatting
