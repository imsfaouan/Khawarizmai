## What is Pinokio?

**Pinokio** is a smart browser designed specifically to install and run open-source AI tools — with one click.

Instead of opening a Terminal and typing commands, or installing Python and Git and dealing with compatibility issues — Pinokio handles all of this under the hood. All you see is an "Install" button, then "Run".

The project has reached **7,300 stars on GitHub** — and keeps growing with every new AI tool that appears.

## How Does Pinokio Work?

The idea is simple and brilliant at the same time:

Pinokio contains everything you need inside it — Python, Git, Conda, pip, npm, and other tools — all installed and isolated in one folder on your machine `~/pinokio`.

When you press Install on any tool:
- Pinokio automatically downloads the code from GitHub
- Installs all requirements in an isolated environment
- Runs the tool without affecting any other programs on your device

And when you delete a tool — all its files disappear completely without any leftovers.

## Installation — Only 3 Steps

**Step 1 — Download Pinokio:**

Go to the releases page and download the version for your operating system:
[Pinokio v7.2.0 Releases](https://github.com/pinokiocomputer/pinokio/releases/tag/v7.2.0)

**For Windows:**
Download the `.exe` file and run it directly

**For Mac:**
Download the `.dmg` file (Intel or Apple Silicon)

**For Linux:**
Download the `.AppImage` or `.deb` file depending on your distribution

**Step 2 — Open Pinokio:**
After installation, open the app directly — you'll find a simple welcome page.

**Step 3 — Choose a tool and install it:**
Go to the **Discover** tab and search for any tool you want — press Install and wait.

That's it. No Terminal, no Python, no problems.

## Security — How to Protect Yourself?

Pinokio is designed with security from the ground up:

**Isolated programs** — Everything inside `~/pinokio`, doesn't touch the rest of your system.

**Open-source scripts** — Every script is written in JSON and you can read it before running.

**Safe Discover page** — Programs shown there are personally reviewed by the team before being added.

**Supply Chain Protection** — Prevents installing packages published less than 72 hours ago to avoid new vulnerabilities.

> **Warning:** Do not install scripts from untrusted external sources. Stick to the Discover page for complete security.
